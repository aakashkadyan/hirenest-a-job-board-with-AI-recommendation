# 💼 HireNest

A modern job portal connecting employers and job seekers with intuitive job management tools.

---

## 🚀 Features

### 👤 Job Seekers

- Register, log in, and create profiles
- Upload resume for automatic profile creation (using resume parsing)
- Search jobs using filters (keywords, location, skills, salary)
- Apply for jobs and track application history
- Receive personalized job recommendations
- Bookmark jobs for later
- Receive email notifications about matching jobs and application updates

### 🏢 Employers

- Register, log in, and manage job postings
- Post new jobs with detailed criteria
- Edit/delete jobs as needed
- Manage received job applications


---

## 📋 Functional Requirements

### Job Posting and Management

- Employers can create an account, post jobs, edit, delete, and manage applications.

### Job Search and Application

- Job seekers can register, search with filters, apply to jobs, track applications, and save/bookmark jobs.

### Email Notifications

- For job matches and application status updates
---

## ⚙️ Non-Functional Requirements

- **Scalability**: Horizontal backend scalability
- **Security**: RBAC, data encryption, and HTTPS
- **Usability**: Fully responsive and mobile-friendly

### 🎯 Job Seekers

- **Database**: MongoDB
- **Authentication**: JWT

## 🚀 Installation

1. Clone the repository and navigate into it:
   ```bash
   git clone https://github.com/your-username/hirenest-a-job-board-with-AI-recommendation.git
   cd hirenest-a-job-board-with-AI-recommendation
2. Install dependencies for both frontend and backend:
   
   cd backend && npm install
   
   cd ../frontend && npm install
   
4. Set up .env files in both frontend and backend folders with required environment variables.
5. Start the development servers:
   cd ../backend && npm run dev
   
   cd ../frontend && npm start


## 🛡️ Security

- Token-based authentication (JWT)
- HTTPS for secure communication
